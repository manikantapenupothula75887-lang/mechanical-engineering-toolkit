# mechanical-engineering-toolkit
A responsive Mechanical Engineering Toolkit web application featuring engineering calculators, material property database, formula library, unit converters, stress-strain analysis, beam deflection tools, and interactive engineering charts for students, educators, and professionals.
##  Live Demo
https://manikantapenupothula75887-lang.github.io/mechanical-engineering-toolkit/
##  Features

* Engineering Calculators

  * Stress Calculator
  * Torque Calculator
  * RPM Calculator
  * Pressure Calculator
  * Heat Transfer Calculator
  * Power Calculator
  * Efficiency Calculator

* Engineering Analysis

  * Stress-Strain Curve Visualization
  * Beam Deflection Analysis
  * Interactive Engineering Charts

* Reference Tools

  * Formula Library
  * Material Property Database
  * Engineering Domain Explorer

* Utilities

  * Unit Converter
  * PDF Report Generation
  * CSV Export
  * Dark/Light Mode

##  Technologies Used

* HTML5
* CSS3
* JavaScript
* Chart.js
* MathJax
* jsPDF

##  Responsive Design

The application is fully optimized for:

* Desktop Computers
* Tablets
* Mobile Devices
